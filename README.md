# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python that analyzes resumes, extracts skills, compares them with required job skills, and provides a resume match score along with improvement suggestions.

---

## 🚀 Features

- 📂 Upload Resume (PDF)
- 📄 Extract text from resumes
- 🧠 Identify technical skills automatically
- 📊 Calculate Resume Match Score
- 💡 Suggest missing skills
- ⚡ Fast and easy to use

---

## 🛠️ Technologies Used

- Python
- Streamlit
- PyPDF2
- Pandas
- Regular Expressions (Regex)
- CSV

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/AI-Resume-Analyzer.git
```

Go to the project folder

```bash
cd AI-Resume-Analyzer
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

## 📸 How It Works

1. Upload your resume (PDF).
2. The application extracts the resume text.
3. Skills are identified from the resume.
4. The extracted skills are compared with the required skills.
5. A Resume Match Score is generated.
6. Suggestions are displayed to improve the resume.

---

## 📂 Dataset

The project uses:

- `skills.csv` – List of technical skills used for matching.
- `sample_resume.pdf` – Sample resume for testing.
